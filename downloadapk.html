VPN အသုံးပြုရန် အောက်ပါ အဆင့်များကို အစဉ်လိုက် လုပ်ဆောင်ပါ

၁

APK ကို ဒေါင်းလုတ်ဆွဲပါ

အထက်ပါ APK ဒေါင်းလုဒ် ခလုတ်ကို နှိပ်၍ ဖုန်းထဲသို့ ဒေါင်းလုဒ်ရယူပါ။

၂

APK ထဲဝင်ပါ

အက်ပလီကေးရှင်းကို Install လုပ်ပြီး ဖွင့်ပါ။ (အင်တာနက်ခဏလိုမည်)

၃

မျဉ်းသုံးကြောင်းကို နှိပ်ပါ

အက်ပလီကေးရှင်း၏ ဘယ်ဘက်ထိပ်ရှိ မျဉ်းသုံးကြောင်း (Menu) ထဲသို့ ဝင်ပါ။

၄

Membership ကို နှိပ်ပါ

Menu ထဲမှ Membership ဆိုသော အချက်အလက်နေရာကို နှိပ်ပါ။

၅

Device ID ကို Copy ယူ ပို့ပေးပါ

ပေါ်လာသော Device ID ကို Copy ယူ၍ အက်ဒမင်ထံ သို့ ပို့ပေးပါ။

၆

Login နှိပ်ပါ

Admin က ရပြီဟု အကြောင်းပြန်လျှင် ညာဘက်ထိပ်ရှိ Login ကို နှိပ်၍ စတင်သုံးစွဲပါ။
